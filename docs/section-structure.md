# Structural Rules

Each section must:
• Be isolated component
• Follow consistent spacing
• Use structured grid layout
• Avoid overcrowding

Pages only assemble sections.

No inline styling.
No random layout deviation.

---

---

# Implementation Rules for Sections

Each section must:

• Use Ant Design components where applicable
• Use Tailwind for layout and spacing
• Avoid raw HTML form controls
• Keep UI consistent across pages

Example:

CTA Section:
Use <Button /> from antd.

Contact Page:
Use <Form />, <Input />, <Button /> from antd.

ROI Section:
Use <Statistic /> for numbers.
